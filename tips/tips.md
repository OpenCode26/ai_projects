Overview
========

Prompting tips to ask the AI Agents to write the Project requirement after developed something from the scratch and then you can develop it further.

example:

1. Please now write me a comprehensive tutorial in markdown called tutorial.md that is suitable for a complete beginner in frontend config, to walk me through what you have done here. Include a summary of the technology, high level walk through, and a detailed code review with code samples. End with 5 suggestions for way the code could be improved based on a self-review.

2. Please do a comprehensive code review of this project and write the results to review.md including any remedial action needed. Don't  actually change any code.
